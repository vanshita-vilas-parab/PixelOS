01 — Object Detection

YOLOv8.
Real-time. Every frame.

detect.py runs Ultralytics YOLOv8 on each camera frame, detecting vehicles and obstacles with bounding boxes and heuristic depth estimates from bounding-box height. Annotated frames stream directly to the Flask web dashboard at localhost:5000.

01

02 — Lane Detection

Canny edges.
Hough lines. Live alerts.

lane.py applies Canny edge detection and Hough line transforms on every incoming frame to track lane geometry. When the vehicle drifts outside detected boundaries, the dashboard triggers an immediate visual lane-keeping warning.

02

03 — Traffic Sign OCR

Indian speed limits.
Read locally. No cloud.

sign_ocr.py uses Tesseract-based OCR to parse Indian speed-limit signs from the video feed. Recognised limits are displayed live on the glassmorphic dashboard overlay — fully offline, zero API dependency.

03

04 — Robot Eye UI

A co-pilot that
reacts with you.

The Web-Eye-Animation widget is embedded directly into the dashboard as a personality-driven virtual assistant. Its animated eyes respond to lane deviation events and ADAS system states in real time — groundwork for a full Groq/OpenAI conversational co-pilot.
